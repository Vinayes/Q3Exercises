My First Bootstrap Page

HTML

HyperText Markup Language

HyperText Markup Language (HTML) is a code used to create and structure the content on a website, such as text, images, and links. It defines the structure and layout of a webpage.

CSS

Cascading Style Sheets

Cascading Style Sheets (CSS) is a code used to define the appearance of a webpage, such as layout, colors, and fonts. It separates the presentation of a website from its structure, allowing for easy visual changes to be made across multiple pages at once.

Bootstrap

Bootstrap is a free and open-source framework for developing responsive and mobile-first web pages. It provides a set of pre-designed components such as navigation bars, forms, buttons, and modals, which can be easily customized to create a consistent and responsive design.
